# Demo Input: Membership Tier Benefits

## Scenario Type

Membership Tier Benefits

## Input Description

This demo represents 6 fictional screenshots from a loyalty program mini app page. The screenshots include:

- Membership enrollment rules
- Tier upgrade rules
- Points rules
- Birthday benefits
- Tier benefits
- Upgrade gifts
- Points arrival rules

No real screenshots are included. This demo uses anonymized mock OCR text reconstructed from public-style loyalty program screenshots.

## Anonymization Rules

- Brand name: `Brand A`
- Loyalty program name: `Brand A Loyalty Program`
- Channel name: `Official Mini Program`
